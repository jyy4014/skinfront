# 피부 분석 이미지 해상도 가이드라인

## 📊 권장 해상도 기준

### 최소 해상도
- **600px 이상**: 기본적인 얼굴 인식 및 피부 분석 가능
- **700px 이상**: 권장 최소 해상도 (현재 설정값)
- **800px 이상**: 이상적인 최소 해상도

### 최적 해상도
- **1000-1200px**: 피부 세부 분석 (모공, 색소, 주름)에 최적
- **1024px**: 현재 시스템 최대 처리 크기 (MAX_WIDTH)

### 최대 해상도
- **2000px 이상**: 처리 시간 증가, 비용 증가, 정확도 향상은 미미
- **4000px 이상**: 불필요하게 큰 해상도, 권장하지 않음

---

## 🎯 분석 목적별 권장 해상도

### 1. 얼굴 인식 (Face Detection)
- **최소**: 200-300px
- **권장**: 400-600px
- **목적**: 얼굴이 있는지, 정면인지 확인

### 2. 기본 피부 분석 (Basic Skin Analysis)
- **최소**: 600-700px
- **권장**: 800-1000px
- **목적**: 전체적인 피부 상태, 색소, 홍조 등

### 3. 세부 피부 분석 (Detailed Skin Analysis)
- **최소**: 800px
- **권장**: 1000-1200px
- **목적**: 모공, 미세 주름, 색소 침착 등 세부 분석

### 4. 의료급 정밀 분석 (Medical-Grade Analysis)
- **최소**: 1200px
- **권장**: 2000px 이상
- **목적**: 매우 정밀한 피부 상태 분석 (현재 시스템에서는 불필요)

## 💡 실용적인 권장사항

### 모바일 카메라 기준
- **일반 스마트폰**: 1000-2000px (충분함)
- **고해상도 카메라**: 2000-4000px (리사이즈 권장)

### 현재 시스템 기준
- **최소 허용**: 700px (MIN_WIDTH_FOR_ANALYSIS)
- **최대 처리**: 1024px (MAX_WIDTH)
- **품질 검사**: 512px로 리사이즈하여 수행

### 비용 및 성능 고려
- **600-800px**: 비용 효율적, 충분한 품질
- **800-1200px**: 균형잡힌 선택 (권장)
- **1200px 이상**: 비용 증가, 성능 저하, 정확도 향상은 미미

## 📝 결론

### 일반적인 피부 분석 앱 기준
1. **최소**: 600-700px (기본 분석 가능)
2. **권장**: 800-1000px (균형잡힌 선택)
3. **이상적**: 1000-1200px (세부 분석 가능)

### 현재 시스템 설정
- **MIN_WIDTH_FOR_ANALYSIS: 700px** ✅ 적절함
- **MAX_WIDTH: 1024px** ✅ 적절함

### 사용자 권장사항
- **700px 이상**이면 분석 가능
- **800px 이상**이면 더 정확한 분석
- **1000px 이상**이면 최적의 품질

**결론**: 700px도 충분하지만, 800-1000px이 가장 균형잡힌 선택입니다.
