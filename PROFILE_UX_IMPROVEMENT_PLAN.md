# 프로필 UX 개선 기획서

## 🎯 목표
사용자가 프로필을 쉽고 즐겁게 관리할 수 있도록 직관적이고 친화적인 UX 제공

## 📊 현재 상태 분석

### 현재 프로필 페이지 구조
1. **프로필 메인** (`/profile`)
   - 기본 정보 표시
   - 분석 기록 목록
   - 로그아웃 버튼

2. **프로필 편집** (`/profile/edit`)
   - 폼 기반 편집
   - 비밀번호 변경

3. **프로필 완성** (`/profile/complete`)
   - 피부 타입 선택
   - 관심사 선택

### 개선 필요 사항
- ❌ 통계/인사이트 부족
- ❌ 프로필 완성도 시각화 부족
- ❌ 빠른 액션 버튼 부족
- ❌ 친절한 안내 메시지 부족
- ❌ 단계별 가이드 부족

## 🎨 개선 계획

### 1. 프로필 메인 페이지 개선

#### 1.1 프로필 헤더 카드
- **프로필 아바타/이름**
- **프로필 완성도 원형 차트** (0-100%)
- **빠른 통계** (분석 횟수, 최근 분석일)
- **"프로필 완성하기" CTA** (완성도 < 100%일 때)

#### 1.2 통계 카드 섹션
- **분석 통계**
  - 총 분석 횟수
  - 이번 달 분석 횟수
  - 마지막 분석일
- **개선 추이**
  - 평균 점수 변화
  - 가장 개선된 항목
- **시술 추천**
  - 관심 등록한 시술 수
  - 최근 추천 시술

#### 1.3 빠른 액션 버튼
- **"새 분석 시작하기"** (그라데이션 버튼)
- **"프로필 완성하기"** (완성도 < 100%)
- **"히스토리 전체 보기"**
- **"설정"**

#### 1.4 최근 활동
- 최근 분석 기록 (3-5개)
- "더 보기" 링크

### 2. 프로필 편집 페이지 개선

#### 2.1 단계별 가이드
- **Step 1: 기본 정보** (이름, 별명, 생년월일)
- **Step 2: 연락처** (전화번호, 국적)
- **Step 3: 피부 정보** (피부 타입, 관심사) - `/profile/complete`로 이동

#### 2.2 실시간 검증 피드백
- ✅ 필드별 실시간 검증
- ✅ 친절한 에러 메시지
- ✅ 완성도 표시

#### 2.3 저장 전 확인
- 변경사항 요약 모달
- "저장하시겠습니까?" 확인

### 3. 프로필 완성도 시각화

#### 3.1 원형 진행률 차트
- 0-100% 시각화
- 색상 코딩 (빨강 → 주황 → 노랑 → 초록)
- 애니메이션 효과

#### 3.2 완성도 상세
- 완성된 필드 목록
- 미완성 필드 목록
- 각 필드별 완성 버튼

### 4. 사용자 친화적 메시지

#### 4.1 환영 메시지
- 첫 방문 시: "환영합니다! 프로필을 완성해주세요"
- 완성도 낮을 때: "프로필을 완성하면 더 정확한 분석을 받을 수 있어요"
- 완성도 높을 때: "거의 완성되었어요! 조금만 더 채워주세요"

#### 4.2 성취감 부여
- 완성도 달성 시 축하 메시지
- 마일스톤 배지 (50%, 80%, 100%)

## 🛠 구현 우선순위

### P0 (즉시 구현)
1. ✅ 프로필 완성도 원형 차트
2. ✅ 통계 카드 (분석 횟수, 최근 분석일)
3. ✅ 빠른 액션 버튼
4. ✅ 친절한 안내 메시지

### P1 (다음 단계)
1. 개선 추이 그래프
2. 단계별 편집 가이드
3. 저장 전 확인 모달
4. 마일스톤 배지

## 📱 디자인 원칙

1. **명확성**: 모든 정보가 명확하게 표시
2. **접근성**: 키보드 네비게이션, 스크린 리더 지원
3. **피드백**: 모든 액션에 즉각적인 피드백
4. **일관성**: 디자인 토큰 사용
5. **친절함**: 따뜻하고 격려하는 톤
